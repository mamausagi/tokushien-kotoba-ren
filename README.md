# ことばれんしゅう（kotoba-ren）

発達障害児の言語訓練を「楽しく継続」するためのPWA。**2語文 → 3語文 → やりとり → 会話**へのステップアップを支援します。就学前〜小学校低学年の支援級児童向け。

🔗 公開URL: https://mamausagi.github.io/tokushien-kotoba-ren/

## 3つのモード
- **🧩 ことばをつくろう** — 絵カードを並べて文を作る。レベル＝文の型（なに＋どうする／だれ＋どうする／どんな＋なに／3語文）。完成すると助詞を補った自然文を読み上げ。
- **🦜 まねっこ** — お手本を聞いて模倣。録音して聞き比べ、音声認識でほめる。
- **💬 しつもんあそび** — アプリの質問に絵で答える会話ターン練習。

## 音声の工夫
- **かぞくの こえ**：父母・祖父母などの声プロファイルを作り、単語ごとに録音 → 子どもがその声で再生（録音が無ければ機械音声=TTSに自動フォールバック）。馴染みの声は反応・模倣が出やすい。
- 子どもの**自分の声の録音再生**＋音声認識ほめ。
- 録音音声は**端末内（IndexedDB）のみに保存**。サーバーへは一切送信しません。

## 設定（おとなモード）
⚙️ を長押し → ⭐3回タップで開く。音ON/OFF・読み速度・機械音声の選択・かぞくの声の録音管理・写真カード追加・⭐リセット。

## 技術メモ
- 単一HTML・完全オフライン動作（PWA）。データは localStorage（設定・写真カード・⭐）＋ IndexedDB（録音音声）。
- マイク録音は HTTPS（GitHub Pages）または localhost でのみ動作。

## 使い方（iPad/iPhone）
Safari で公開URLを開く → 共有 → 「ホーム画面に追加」で全画面アプリとして起動。
